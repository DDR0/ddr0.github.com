Linux Quick-Guide

Note: Arch Linux users may not be able to compile using the supplied makefile. Don't feel bad. It's not you, it's me.

Steps

① Run, in a console, "sudo apt-get install ccache libboost-dev libboost-regex-dev libboost-system-dev libglew1.5-dev libsdl-image1.2-dev libsdl-mixer1.2-dev libsdl-ttf2.0-dev libsdl1.2-dev libz-dev libpng12-dev". This will install the dependancies of the game.

② If you are not already in the Cube Trains folder (it'll have a file called "makefile"), navigate there. ('cd' to navigate, 'ls' to list files.)

③ Next, run the "make" command. This should compile the game. If it doesn't say "ERROR" somewhere, then all is good. (It will print a bunch of stuff like "ccache g++ -DIMPLEMENT_SAVE_PNG -fno-inline-functions -g -O2 -fno-inline-functions bla bla bla". Don't worry, that means it's working.) If you get an error and don't know what to do, leave us a note on the forums and we'll figure out what's wrong. Don't be shy, that's what they're there for.

④ Run the game with "./game". If everything is working right, you should see the game start up. Have fun!
