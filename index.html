CUBE TRAINS

Solve routing puzzles in a dense city with Cube Trains. You start off as a young urban planner trying to connect a city's fragmented rail lines together, and work your way up to become an old urban planner trying to connect a city's fragmented rail lines together! Over the course of your career, you'll build towering structures, beautiful bridges, and some truly convoluted infrastructure. Download the beta below, for free. Also, pictures. Coming summer 2012. Yours for the price of two Canadian dollars.

Features:

  • ☑ Puzzles in three dimensions.
  • ☐ Twenty reasonably unique levels.
  • ☑ Undo/redo.
  • ☑ Individual level saving.
  • ☑ Colourblind mode.

Release notes: Beta releases available for download. There is currently no sound or reset button, and inconsistent art quality. This does not affect mac users, but there is a bug with regards to the save file which manifests itself by Cube Trains refusing to finish starting up or crashing after rendering the first frame. This is caused by a Frogatto and Friends save file in the same place Cube Trains is expecting one of its saved games. To remedy this problem, find (search for) a file named "save.cfg". It will probably be in a folder called .frogatto or something like that. Remove the file from the folder. Once Cube Trains has started up, rename either file and put the first file back in the folder. This will be fixed in a day or two in version 0.2.1 or greater.
